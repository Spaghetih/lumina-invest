Welcome to the source code of **Lumina Invest**, a modern, premium stock portfolio tracking application designed with a sleek Glassmorphism UI, smooth animations, and a cohesive dark mode experience.

### ✨ Key Features
- **Live Market Tracking:** Real-time stock quotes and portfolio valuation powered by the Yahoo Finance API.
- **Dividend Calendar:** Automatically estimates your annual passive income and provides a detailed chronological timeline of upcoming and past payouts. Features dynamic global company logo resolving.
- **Integrated AI Assistant:** A built-in financial advisor powered by OpenAI. It analyzes your current portfolio, identifies sector exposure, and provides actionable diversification recommendations.
- **Easy Import:** Seamlessly import your transaction history directly from brokers like Revolut.
- **Advanced Analytics:** Smooth historical performance charts, automatically calculated All-Time Highs (ATH), and detailed sector breakdowns.

### 🛠️ Tech Stack
- **Frontend:** React.js, Vite, Vanilla CSS (Extensive custom properties & animations)
- **Backend:** Node.js, Express (Proxy server for external APIs)
- **Data APIs:** Yahoo Finance, OpenAI API, Parqet Logos
- **Architecture:** Secure local storage (`portfolio.json`)

### 🚀 Getting Started
1. Clone the repository
2. Run `npm install` in both the main directory and any backend directories
3. Start the Vite dev server with `npm run dev`
4. Start the backend proxy with `node server.js`
